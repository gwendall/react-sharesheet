# react-sharesheet Example

A Next.js example app demonstrating the `react-sharesheet` component library.

## Getting Started

First, make sure you've built the library:

```bash
# From the root directory
pnpm build
```

Then install dependencies and run the example:

```bash
cd example
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the demo.

## What's Included

This example demonstrates:

- **Basic usage** - Simple share button with ShareSheetDrawer
- **Filtered platforms** - Using `show` prop to display only specific platforms
- **Custom labels** - Customizing button labels with the `labels` prop
- **Preview images** - Adding a preview image to the share sheet
